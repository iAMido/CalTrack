# CalTrack — מדריך שימוש

## מה זה CalTrack?
בוט טלגרם חכם למעקב קלוריות אישי. מצלמים אוכל, הבינה המלאכותית מזהה את המרכיבים, מאשרים משקלים — והמערכת עוקבת אחרי הקלוריות והמאקרו שלך אוטומטית.

---

## תחילת עבודה
פתח שיחה עם הבוט בטלגרם ושלח פקודה כלשהי. הבוט מוגדר עבורך בלבד — אף אחד אחר לא יוכל להשתמש בו.

---

## 📷 תיעוד ארוחה — הדרך הראשית

### 1. שלח תמונה של האוכל
פשוט צלם את הצלחת ושלח לבוט. הבינה המלאכותית תזהה את כל המרכיבים בתמונה.

### 2. בדוק את התוצאה
הבוט יחזיר רשימה כזו:
```
🌙 ארוחת ערב — 02 מאי, 23:17

1. tuna (טונה)
   הערכת AI: 100g (ביטחון: 70%)

2. avocado (אבוקדו)
   הערכת AI: 70g (ביטחון: 60%)

──────────────────
סה"כ: ~350 קל׳
```

### 3. כפתורי הפעולה לכל מרכיב
לכל מרכיב יש שורת כפתורים:

| כפתור | מה עושה |
|--------|---------|
| `80g / 100g / 120g` | בחר משקל מוצע |
| `✏️` | הקלד משקל ידני (כתוב מספר בגרמים) |
| `🔄` | תקן שם שגוי — הקלד את השם הנכון |

### 4. כפתורים כלליים

| כפתור | מה עושה |
|--------|---------|
| `✅ Confirm All` | שמור את כל הארוחה |
| `❌ Cancel` | בטל ואל תשמור |
| `🔄 Re-analyze` | שלח מחדש לניתוח |
| `➕ Missing item?` | הוסף מרכיב שהבינה המלאכותית פספסה |

### 5. הוספת מרכיב חסר
אם הבוט פספס משהו (למשל שמן זית), לחץ `➕ Missing item?` ואז כתוב:
```
שמן זית, 15
```
*(שם המרכיב, פסיק, גרמים)*

### 6. תיקון שם שגוי
אם הבינה המלאכותית זיהתה "אגס" אבל זה בעצם "תפוח" — לחץ 🔄 ליד אותו פריט וכתוב:
```
תפוח
```
הבוט יחשב מחדש את הקלוריות.

---

## 🏷 סריקת תווית תזונה — `/label`

לסריקת מוצר ארוז (כגון יוגורט, דגני בוקר וכו'):

1. שלח `/label`
2. הבוט יבקש תמונה — צלם את **טבלת הערכים התזונתיים** בגב האריזה
3. הבינה המלאכותית תחלץ קלוריות, חלבון, פחמימות ושומן לכל 100 גרם
4. המוצר נשמר במסד הנתונים — בפעם הבאה שתצלם אותו, הוא יזוהה אוטומטית

**טיפ:** וודא שעמודת "ל-100 גרם" גלויה בצילום.

---

## ➕ הוספה ידנית — `/add`

להוספת מרכיב שנשכחת לתעד, אחרי שהארוחה כבר נשמרה:

```
/add צהריים 15 גרם שמן זית
/add ערב 200 גרם אורז לבן
/add 120 גרם סלמון
```

**פורמט:** `/add [ארוחה] [כמות] גרם [שם המאכל]`

- **ארוחה** (אופציונלי): בוקר / צהריים / ערב / חטיף
- אם לא מציינים ארוחה, הבוט יחליט לפי שעה
- ניתן לכתוב **בעברית** — הבוט יתרגם אוטומטית

---

## ⚖️ תיעוד משקל גוף — `/weight`

```
/weight 87.3
```

מתעד את משקל הגוף הנוכחי ומעדכן את חישובי הקלוריות.

---

## 💧 תיעוד שתייה — `/water`

```
/water 500
```

מתעד 500 מ"ל שתייה. ניתן לשלוח כמה פעמים ביום — מצטבר.

---

## 🏃 תיעוד ריצה — `/run`

```
/run 5.2 28:30 152
```

**פורמט:** `/run [ק"מ] [דקות:שניות] [דופק ממוצע]`

- דופק — אופציונלי
- הבוט מחשב קלוריות שנשרפו לפי קצב הריצה

---

## 📊 סיכום יומי — `/summary` או `/s`

מציג את כל נתוני היום:
- סה"כ קלוריות שנאכלו
- פירוט מאקרו (חלבון / פחמימות / שומן / סיבים)
- קלוריות שנשרפו בפעילות גופנית
- יעד יומי ועודף/חוסר
- שתייה ומשקל

## 🔧 כיול מחדש — `/calibrate`

מחשב מחדש את ה-BMR וה-TDEE שלך לפי המשקל הנוכחי ומעדכן את יעד הקלוריות היומי.

**מומלץ** להפעיל לאחר כל שינוי משקל משמעותי.

---

## ❓ עזרה — `/help`

מציג את כל הפקודות הזמינות.

---

## 💡 טיפים לשימוש מיטבי

1. **צלם מקרוב** — ככל שהתמונה ברורה יותר, הזיהוי מדויק יותר
2. **תמיד אשר משקלים** — הבוט מציע הערכה, אתה יודע הכי טוב
3. **השתמש ב-🔄** כשהבוט מזהה את המאכל בצורה שגויה
4. **סרוק תוויות** של מוצרים חדשים — הם נשמרים לצמיתות
5. **כתוב בעברית** — כל הפקודות עובדות גם בעברית
6. **הוסף שמן/רטבים** עם `/add` — הם לרוב לא נראים בתמונה
